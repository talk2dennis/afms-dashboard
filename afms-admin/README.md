# AFMS Admin Frontend

This folder contains the React + TypeScript admin dashboard for AFMS (Automated Flood Monitoring System), developed as part of an MSc Information Technology project.

## Related Repositories

- Mobile app (user-facing): `https://github.com/talk2dennis/afms`
- Backend API enabled: `https://github.com/talk2dennis/floodmap`

Both repositories are part of the same MSc project and connect to a shared backend API.

## Quick Start

```bash
npm install
npm run dev
```

## Environment

Create/update `.env` with:

```env
VITE_API_BASE_URL=http://localhost:5000/api/
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`

For full project documentation (features, architecture, API summary, and deployment notes), see the repository root README: `../README.md`.
